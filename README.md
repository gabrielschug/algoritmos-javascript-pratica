# 🧠 Lógica de Programação e Algoritmos com JavaScript

Este repositório documenta minha jornada de consolidação dos fundamentos de programação, resolvendo problemas práticos e criando pequenos projetos baseados no livro *"Lógica de Programação e Algoritmos com JavaScript"* (Edécio Fernando Iepsen).

O foco principal é **desenvolver o **raciocínio lógico****, **manipulação do DOM** e a **estruturação de algoritmos eficientes** aplicados em interfaces web reais.

## 🛠️ Tecnologias e Conceitos Aplicados

* **JavaScript (ES6+):** Estruturas de controle (if/else, switch), laços de repetição (for, while), manipulação de arrays e objetos, funções, eventos e manipulação do DOM.
* **HTML5:** Estruturação semântica de formulários e páginas.
* **CSS3:** Estilização de interfaces para melhorar a experiência do usuário.
* **Lógica Computacional:** Resolução de problemas matemáticos, algoritmos de ordenação, pesquisa e estruturação de dados.

---

## 📂 Estrutura do Repositório

Os projetos estão organizados de forma progressiva, do básico ao avançado:

* **Capítulo 01:** Integração HTML + JS, variáveis e operações matemáticas.
* **Capítulo 02:** Condições (if/else), operadores lógicos e ternários.
* **Capítulo 03:** Laços de repetição (for, while, do/while).
* **Capítulo 04:** Arrays (vetores) e métodos de manipulação.
* **Capítulo 05:** Objetos e manipulação avançada.
